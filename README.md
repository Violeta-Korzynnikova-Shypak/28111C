# Página principal — Fórmula 1

Este repositorio contiene una página web estática de ejemplo sobre Fórmula 1.

Archivos principales:

- `index.html` — Página principal (en español) con secciones: noticias, equipos, pilotos, calendario y resultados.
- `styles.css` — Estilos y diseño responsivo para la página.

Cómo abrir la página localmente:

1. Abrir `index.html` directamente en tu navegador (doble clic o abrir archivo).
2. O bien, servir el directorio con un servidor sencillo (ejemplo con Python):

```bash
python3 -m http.server 8000
# y luego abrir http://localhost:8000 en el navegador
```
